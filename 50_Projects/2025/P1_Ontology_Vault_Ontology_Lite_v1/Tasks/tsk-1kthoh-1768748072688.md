# Ontology Lite v1 - edge_types 사전 정의 및 예시 작성

## 목적

LLM agent가 관계를 "추론"하지 않도록, 관계 타입을 의미적으로 고정한다.

## 해야 할 일

- edge_types 정의(자연어 1~2줄)
- from/to 제약, cardinality(1:1/1:N), cycle 허용 여부
- 예시(실제 엔티티 1~2개씩)
- 금지 규칙(중복 의미 edge 방지) + "애매할 때 선택 규칙"
