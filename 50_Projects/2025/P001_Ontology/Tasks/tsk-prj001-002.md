# Episode 엔티티 검증

> Task ID: `tsk:001-02` | Project: `prj:001` | Status: pending

## 목표

**완료 조건**:
1. Episode가 2-4시간 위험 구간 캡처 가능 확인

3. 폭식 전조 패턴 Episode로 표현 가능

## 상세 내용

### 배경
Episode는 여러 Event를 묶어 의미 있는 시간 구간을 표현. 폭식 전조 패턴(anxiety → emptiness → urge → binge) 캡처에 핵심.

### 작업 내용
1. Episode 스키마 정의 검토
2. Event 그룹핑 규칙 정의
3. 실제 패턴 데이터로 Episode 생성 테스트
4. 재현율 측정
