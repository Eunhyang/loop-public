## Tech Spec

### 수정 파일 1: `app/performance/data/dummy-performance.ts`

#### STATUS_BADGE_CONFIG 수정

**Label 변경:**
- `expansion_failed.label`: "확장 실패" -> "성장 둔화"

**Description 추가/개선:**

| Status | Description |
|--------|-------------|
| `early_success` | CTR 8%+, 시청률 30%+, 7일간 3배 이상 확장. 알고리즘이 적극 추천 중 |
| `stable` | 일정한 조회수 유지. 장기 트래픽 소스로 활용 가능 |
| `expansion_failed` | 7일 조회수가 24시간의 3배 미만. 알고리즘 추천이 정체된 상태로, 주제/타이밍 점검 필요 |
| `exposure_ok_click_weak` | CTR 5% 미만. 썸네일/제목이 클릭을 유도하지 못함 |
| `click_ok_watch_weak` | 평균 시청 30% 미만. 초반 10초 훅 점검 필요 |
