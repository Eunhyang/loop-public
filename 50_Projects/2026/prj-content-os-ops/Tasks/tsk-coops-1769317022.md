# Content OS - Explorer 검색결과 상세 모달

> Task ID: `tsk-coops-1769317022` | Project: [[prj-content-os-ops]] | Status: todo

---

## 한 줄 정의

Explorer 검색 결과 리스트에서 특정 영상 클릭 시 **오버레이 상세 모달**이 열리고, 영상의 핵심 KPI(Impact/Velocity/노출도) + 콘텐츠 메타 + 채널 정보 + 추천까지 한 화면에서 판단/액션(보관/차단)을 수행한다.

---

## 목적 & 성공기준

### 목적
- 검색 결과를 "리스트 탐색" 상태로 유지하면서 상세 판단에 필요한 정보 즉시 제공
- 의사결정(보관/차단/분석 우선순위)을 Impact/Velocity 중심으로 빠르게 내리도록 UX 재구성

### 성공기준
- 상세 모달 오픈율 증가
- 모달 내 보관/차단 전환율 증가
- 모달 오픈 후 체류시간 및 탭 전환율 확보

---

## 범위

### In-Scope
- 검색 결과 테이블에서 상세 모달 오픈/닫기
- 모달 내 KPI(Impact/Velocity/노출도) + 영상/채널/추천 섹션
- 보관/차단 액션
- Impact/Velocity 기반 기여도/성과도 라벨

### Out-of-Scope (v2)
- 별도 상세 페이지 라우팅
- 고급 분석(스크립트 추출, 챕터링, 클립 생성)
- 댓글 전체 목록/감성 분석

---

## 모달 구조

### 헤더 (Sticky)
- 썸네일(좌) + 기본 메타(우): 제목, 업로드일, 조회수, 채널명, 구독자수
- CTA: [YouTube 열기] [보관] [차단]
- 닫기(×) + ESC 지원

### KPI 스트립
1. **Impact** (0~10 스케일) - 참고 가치 점수
2. **Velocity** (vph 또는 views/day) - 일평균 조회수 증가량
3. **노출도** (Bad/Normal/Good/Great)
4. (선택) 조회수 / 업로드 경과일

### 탭 구성 (3개)
1. **영상 정보**: 설명, 태그, 핵심 댓글, 지표
2. **채널 정보**: 채널명, 구독자, 영상 수, 누적 조회수
3. **추천/관련**: 이 채널의 인기 영상 리스트

---

## 기여도/성과도 매핑

### 기여도 (Impact 기반)
| Impact | 라벨 |
|--------|------|
| ≥ 8.0 | 필수분석 |
| 5.0~7.9 | 괜찮음 |
| < 5.0 | 낮음 |

### 성과도 (Velocity 기반)
| 분위수 | 라벨 |
|--------|------|
| 상위 20% | Great |
| 20~50% | Good |
| 50~80% | Normal |
| 하위 20% | Bad |

---

## 인터랙션 규칙

- 오픈: 영상 셀(썸네일/제목) 클릭
- 닫기: X버튼, ESC, 오버레이 클릭
- 추천 영상 클릭: 모달 유지 + 콘텐츠만 교체
- 로딩: 즉시(테이블 데이터) + 지연(상세 데이터)
