### Summary
Pending 페이지에서 Project review 선택 시 Strategic Context가 즉시 표시되도록 React Query 기반 prefetch 구현.

### Changes Made
- `src/queries/keys.ts`: strategicContext query key 추가

### Technical Details
- React Query의 useQueries로 병렬 prefetch

- review mode에서 자동으로 strategic context 펼침
