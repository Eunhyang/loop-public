# Strategy Calendar - 로드맵+캘린더 통합 뷰 구현

## Notes

### Summary

- `dashboard-v2/src/features/strategy/StrategyCalendarPage.tsx` - 통합 페이지
- `dashboard-v2/src/features/strategy/components/roadmap/RoadmapView.tsx` - 로드맵 메인
- `dashboard-v2/src/features/strategy/components/roadmap/QuarterColumn.tsx` - 분기 열
- `dashboard-v2/src/features/strategy/components/roadmap/EntityCard.tsx` - 엔티티 카드
- `dashboard-v2/src/features/strategy/hooks/useRoadmapData.ts` - 데이터 훅
- `dashboard-v2/src/features/strategy/types/roadmap.ts` - 타입 정의

**수정 파일:**

### 기능 상세
- 상단 40%: Q1-Q4 분기별 로드맵 (Track/Hypothesis/Project 카드)
- 하단 60%: 캘린더 (선택된 엔티티의 Task만 필터링 표시)
- Track 선택 시 → 해당 Track의 Project들의 Task 표시
- Hypothesis 선택 시 → validates하는 Project들의 Task 표시
- Project 선택 시 → 해당 Project의 Task만 표시
