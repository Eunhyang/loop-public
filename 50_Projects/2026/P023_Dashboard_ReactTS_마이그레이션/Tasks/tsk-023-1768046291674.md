# Dashboard - Calendar Week View 수직 스크롤 버그 수정

## 설명

### 문제 상황
- Calendar Week View에서 수직 스크롤이 안 됨
- 캘린더가 잘리는 현상

### 원인 분석

#### 근본 원인
`CalendarPage.tsx`의 레이아웃에서 `overflow-hidden`이 중첩되어 스크롤 영역이 생성되지 않음

#### 문제 코드 위치
| 파일 | 라인 | 문제 코드 |
|------|------|----------|
| `CalendarPage.tsx` | 111 | `className="flex h-screen w-full bg-white overflow-hidden"` |
| `CalendarPage.tsx` | 114 | `className="flex-1 flex flex-col h-full overflow-hidden relative"` |

#### 구조 문제
```
<div overflow-hidden>           ← 스크롤 차단
  └─ <div h-full overflow-hidden>  ← 이중 차단 + min-h-0 누락
      ├─ TaskFilterBar (고정)
      ├─ CustomToolbar (고정)
      ├─ FullCalendar (flex-1, height: 100%)  ← 넘쳐도 스크롤 불가
      └─ FilterPanel
```

## Tech Spec

### 수정 파일
`public/_dashboard/dashboard-v2/src/features/calendar/CalendarPage.tsx`

### 수정 내용

#### 1. Line 111 수정
```tsx
// Before
<div className="flex h-screen w-full bg-white overflow-hidden">

// After
<div className="flex h-screen w-full bg-white">
```
- 최상위 `overflow-hidden` 제거

#### 2. Line 114 수정
```tsx
// Before
<div className="flex-1 flex flex-col h-full overflow-hidden relative">

// After
<div className="flex-1 flex flex-col h-full min-h-0 relative">
```
- `overflow-hidden` 을 `min-h-0`으로 변경 (flex child 높이 계산 활성화)

#### 3. CalendarView 래퍼에 스크롤 컨테이너 추가
```tsx
// CalendarView를 감싸는 div에 overflow-auto 추가
<div className="flex-1 min-h-0 overflow-auto">
  <WrapperCalendarView ... />
</div>
```

**원리**:

- 실제 스크롤 컨테이너는 CalendarView 래퍼에 `overflow-auto`로 설정
