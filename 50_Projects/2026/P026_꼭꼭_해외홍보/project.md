# 꼭꼭앱 해외 홍보 (Reddit 2주 파일럿)

## 개요

Reddit 커뮤니티에 매일 꼭꼭앱 홍보글을 작성하여 해외 사용자 반응을 검증하는 2주 파일럿 프로젝트.

## 운영 기간

2026-01-23 ~ 2026-02-05 (14일, 2주)

## 운영 규칙

| 항목 | 내용 |
|------|------|
| 담당자 | 한명학 |
| 빈도 | 매일 1포스트 |
| 대상 커뮤니티 | Reddit (habit tracker, productivity, self-improvement 등) |
| 콘텐츠 유형 | 앱 소개, 사용 후기, 기능 하이라이트 |

## 성공 지표

| KPI | 목표 |
|-----|------|
| 총 포스트 수 | 14개 (매일 1개) |
| 총 Engagement | 100+ (upvotes + 댓글) |
| 앱 다운로드 유입 | 10건+ (Reddit 경유) |

## 태스크 관리

매일 태스크는 템플릿 `tpl-sosi-reddit-promo`로 생성:
- Day 1 ~ Day 14
- 각 태스크에 포스팅 URL, 반응 기록
