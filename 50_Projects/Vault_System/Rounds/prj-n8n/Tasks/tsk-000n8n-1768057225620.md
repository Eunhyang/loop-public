# n8n - Entity Validator Auto-Apply 기능

> Task ID: `tsk-000n8n-1768057225620` | Project: `prj-n8n` | Status: todo

## 목표

Entity Validator 워크플로우에서 자동으로 완성된 필드를 vault에 적용하는 기능 구현.

## 완료 조건

1. Entity Validator 워크플로우가 자동 완성한 필드를 pending review 없이 직접 저장
2. 변경 이력이 audit log에 자동 기록됨
3. Dashboard에서 AUTO-APPLY 뱃지로 표시됨

## 상세 내용

### 현재 문제

- Entity Validator 워크플로우가 생성한 필드들이 pending reviews로만 저장됨
- 확인/승인 프로세스 지연
- Vault 자동화 효율성 저하

### 솔루션

1. **Workflow 개선**: validator 결과를 `/api/tasks/{id}` PATCH로 직접 적용
2. **Audit Trail**: 모든 변경사항을 `_build/audit.log`에 기록
3. **Rollback**: 필요시 이전 버전 복구 가능하도록 git history 활용
