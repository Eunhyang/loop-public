# Strategic Context - 프론트엔드 캐시 조립 구현

## Notes

### Summary
"Strategic Context 보기" 버튼 클릭 시 백엔드 API 호출 없이 dashboard-init 캐시에서 데이터를 조립하여 raw 데이터로 표시하도록 구현.

### Changes Made
- `selectors.ts`: `buildStrategicContext()` 순수 함수 추가
- `queries.ts`: `useStrategicContext`, `usePrefetchStrategicContexts` 훅 수정 (API → 캐시 조립)
- `StrategicContextSection.tsx`: context 탭 제거, raw만 표시
- `api.ts`: `StrategicContextResponse` 타입 수정 (context optional, tasks 추가)

### Commit
- `c9183ec54` - feat(strategic-context): frontend cache assembly instead of API call

### Effect
- Network 탭에서 `/api/projects/{id}/strategic-context` 호출 없음
- tasks (done 포함, _body 포함) 데이터 포함
- 백엔드 API는 LLM 프롬프트용으로 유지
