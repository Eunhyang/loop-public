## Notes

### Summary
Dashboard에 Vault Explorer 기능을 추가하여 public/ 폴더 트리 탐색, 검색, 파일 열기 기능 구현. 여러 파일을 동시에 열고 접기/펼치기/닫기 가능.

### Changes Made

**신규 파일 (13개)**:
- `src/types/vault-explorer.ts` - 서버 DTO 및 UI 타입 정의
- `src/features/vault-explorer/api.ts` - API 래퍼 (folder-contents, file-read, search)
- `src/features/vault-explorer/queries.ts` - React Query hooks
- `src/features/vault-explorer/selectors.ts` - 순수 함수 (sortFolderContents, extractFileName 등)
- `src/features/vault-explorer/components/SearchBar.tsx` - 검색 입력
- `src/features/vault-explorer/components/SearchResults.tsx` - 검색 결과 리스트
- `src/features/vault-explorer/components/FolderTree.tsx` - 루트 폴더 트리
- `src/features/vault-explorer/components/FolderTreeItem.tsx` - 개별 폴더/파일 아이템 (재귀)
- `src/features/vault-explorer/components/VaultExplorerDrawer.tsx` - 왼쪽 drawer 메인
- `src/features/vault-explorer/components/index.ts` - barrel export

- `src/components/common/LeftDrawerShell.tsx` - 왼쪽 drawer 레이아웃 컴포넌트
- `src/components/layout/OpenFilesPanel.tsx` - 열린 파일 패널들

**수정 파일 (5개)**:
- `src/types/index.ts` - vault-explorer export 추가

### 구현 기능
1. **Vault Explorer Drawer**: 헤더 왼쪽 버튼 클릭 → 왼쪽에서 슬라이드 인
2. **폴더 트리**: lazy loading으로 폴더 펼치기/접기
3. **검색**: 2글자 이상 입력 시 vault 검색
4. **파일 열기**: 클릭 → 메인 영역에 패널 추가
5. **접기/펼치기**: CSS로 숨김 (DOM 유지)
6. **닫기**: DOM에서 제거
