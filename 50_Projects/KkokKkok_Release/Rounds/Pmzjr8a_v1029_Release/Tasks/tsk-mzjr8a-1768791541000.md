# sosi - 푸시알림 긴급/중요 알림 채널 적용

## 설명

Android에서 푸시알림을 **긴급/중요 알림 채널**로 전송하여 방해금지 모드에서도 알림이 표시되도록 함.

### 목표
- 식사 리마인더 등 중요한 알림이 사용자에게 확실히 전달되도록 함
- Android 알림 채널의 importance를 HIGH 또는 MAX로 설정

## 체크리스트

- [ ] Android 알림 채널 설정 확인 (importance level)
- [ ] FCM 메시지에 priority: high 설정
- [ ] 실기기 테스트 (방해금지 모드에서 알림 확인)

## 참고

- Android Notification Channel importance levels:
  - IMPORTANCE_MAX: 긴급 알림 (heads-up 표시)
  - IMPORTANCE_HIGH: 중요 알림
  - IMPORTANCE_DEFAULT: 일반 알림
