notes: "## 목표\n\n**아래 UI를 기능없이 UI MVP만 제작** \\\n\\\n0) 톤 가이드 (공통)\n\n- 금지: “폭식”, “위기”,\
  \ “위험”, “통제”, “실패”\n\n- 권장: “패턴”, “자동”, “스위치”, “리셋”, “잠깐 멈춤”, “오늘도 충분히 잘했다”\n\n\
  - 기본 문장 구조: **“지금” + “한 번만” + “10초/1분/10분”** (마찰 낮게)\n\n---\n\n# 1) 온보딩 — 트리거 상황\
  \ 수집 (AI챗 UI)\n\n### Screen: “내 패턴 한 번만 잡아두기”\n\n- 상단 타이틀: **“무의식속 과식하게 되는 순간이 있나요?”**\n\
  \n- 서브: **“딱 한 번만 알려주면, 그 시간에 먼저 톡 쳐줄게요.”**\n\n**Chat Bubble 1**\n\n- 질문: **“보통\
  \ 언제 시작돼요?”**\n\n- 선택칩: `퇴근길` `밤 10~12시` `점심 후` `집 도착 직후` `주말` `기타`\n\n- 하단 힌트:\
  \ **“대충 골라도 괜찮아요.”**\n\n**Chat Bubble 2**\n\n- 질문: **“그때 어디에 있고, 누구랑 있나요?”**\n\n\
  - 선택칩: `버스/지하철` `집` `회사` `차 안` / `혼자` `가족` `친구` `동료`\n\n- 하단 힌트: **“정확하지 않아도 괜찮아요.”**\n\
  \n**Chat Bubble 3**\n\n- 질문: **“보통 뭐부터 하게 되나요?”**\n\n- 선택칩: `배달앱 켠다` `편의점 간다` `냉장고\
  \ 문 연다` `과자 봉지 연다` `기타`\n\n- 하단 힌트: **“‘시작 버튼’ 같은 행동 하나만 골라주세요.”**\n\n**Chat Bubble\
  \ 4**\n\n- 질문: **“그때 몸/마음 상태는요?”**\n\n- 선택칩: `피곤함` `스트레스` `심심함` `습관` `허기` `모르겠음`\n\
  \n- 하단 힌트: **“‘모르겠음’도 아주 좋은 답이에요.”**\n\n**CTA 버튼**\n\n- `다음` (버튼 문구: **“좋아요, 이제\
  \ 한 가지 준비만 더”**)\n\n---\n\n# 2) 온보딩 — “나를 온화하게 만드는 행동” 선택\n\n### Screen: “내 리셋 행동\
  \ 고르기”\n\n- 타이틀: **“그 순간, 가장 쉬운 ‘리셋 행동’은 뭐가 좋아요?”**\n\n- 서브: **“의지 말고 ‘쉬운 것’으로 골라요.\
  \ 1개만 있어도 충분.”**\n\n**선택 리스트 (카드형)**\n\n- `물 한 컵` (추천 배지: **“가장 쉽고 바로 가능”**)\n\n\
  - `1분 스트레칭`\n\n- `1분 책상정리`\n\n- `샤워/세수`\n\n- `스쿼트 10개`\n\n- `반려동물 쓰다듬기`\n\n- `차\
  \ 한 잔`\n\n- `1분 호흡`\n\n**하단 토글(선택)**\n\n- `지금은 하나만 고를래요` (기본 ON)\n\n**CTA**\n\n\
  - 버튼: **“이걸로 갈게요”**\n\n---\n\n# 3) 온보딩 — 시간대/요일 설정 (알림 설정 UI)\n\n### Screen: “그\
  \ 시간에 먼저 알려주기”\n\n- 타이틀: **“자주 자동으로 켜지는 시간, 맞춰볼까요?”**\n\n- 서브: **“완벽하게 맞출 필요 없어요.\
  \ 대략만.”**\n\n**요일 선택**\n\n- `월 화 수 목 금 토 일` (선택된 요일 강조)\n\n**시간 선택**\n\n- Time picker:\
  \ `18:00`\n\n**알림 강도(문구 개선)**\n\n- 기존 “강력알림” 대신:\n\n  - 라벨: **“놓치기 쉬운 시간엔 한 번 더”**\n\
  \n  - 토글: `한 번 더 알려주기` (ON/OFF)\n\n**미리보기 카드**\n\n- **“금요일 18:00쯤, ‘패턴 체크’ 알림이 와요.”**\n\
  \n**CTA**\n\n- 버튼: **“설정 끝”**\n\n- 보조 텍스트: **“언제든 바꿀 수 있어요.”**\n\n---\n\n# 4) 트리거\
  \ 푸시 알림 (모의)\n\n### Push 카드 (OS 스타일)\n\n- 타이틀: **“지금 ‘그 패턴’ 시간이에요”**\n\n- 본문: **“퇴근길에는\
  \ 자동으로 배달앱을 켜기 쉬워요. 10초만 체크할까요?”**\n\n- 버튼: `10초 체크`\n\n(대안 문구 2)\n\n- **“자동 스위치가\
  \ 켜지기 쉬운 시간이에요. 잠깐만 멈춰볼까요?”**\n\n- **“지금은 ‘시작 버튼’을 누르기 쉬운 시간. 먼저 한 번 체크해요.”**\n\n\
  ---\n\n# 5) 체크인 — 갈망/배고픔 점수\n\n### Screen: “지금 상태만 확인”\n\n- 타이틀: **“지금은 어떤 상태예요?”**\n\
  \n- 서브: **“맞히는 게 아니라 ‘알아차리기’만 해요.”**\n\n**입력 1**\n\n- 라벨: **“먹고 싶은 마음(갈망)”**\n\n\
  - 스케일: `0 전혀 아님 … 10 아주 강함`\n\n**입력 2**\n\n- 라벨: **“몸의 배고픔”**\n\n- 스케일: `0 전혀 아님\
  \ … 10 아주 배고픔`\n\n**CTA**\n\n- 버튼: **“다음 (10분 리셋)”**\n\n- 보조: **“여기까지만 해도 반은 끝.”**\n\
  \n---\n\n# 6) 선택 — 온전한 식사 vs 리셋 행동\n\n### Screen: “오늘은 어느 쪽이 더 쉬워요?”\n\n- 타이틀: **“오늘은\
  \ 이렇게 가볼까요?”**\n\n- 서브: **“정답은 없어요. ‘쉬운 쪽’이 정답.”**\n\n**카드 A: 온전한 식사**\n\n- 헤드라인:\
  \ **“온전하게 먹고 끝내기”**\n\n- 서브: **“먹을 거면, 만족 신호까지 가는 방식으로.”**\n\n- 버튼: `이렇게 먹을래요`\n\
  \n**카드 B: 10분 리셋**\n\n- 헤드라인: **“10분만 리셋하고 결정하기”**\n\n- 서브: **“내 리셋 행동: 물 한 컵(또는\
  \ 선택한 행동)”**\n\n- 버튼: `10분 리셋 시작`\n\n(대안 라벨)\n\n- “대체행동” 대신 **“리셋 행동”** / “브레이크”\
  \ 대신 **“잠깐 멈춤”**\n\n---\n\n# 7) 10분 리셋 진행\n\n### Screen: “10분만 같이 가요”\n\n- 타이틀:\
  \ **“좋아요. 10분만 같이 가요.”**\n\n- 서브: **“지금은 ‘결정’이 아니라 ‘리셋’ 시간.”**\n\n**행동 카드**\n\n\
  - **“지금 할 일: 물 한 컵 마시기”**\n\n- 작은 체크: `완료했어요`\n\n**타이머**\n\n- `10:00` 카운트다운 (영상용\
  \ 연출 가능)\n\n**CTA**\n\n- `10분 후 다시 체크`\n\n(대안 문구)\n\n- **“끝까지 잘 못해도 돼요. 시작한 게 이미\
  \ 변화예요.”**\n\n---\n\n# 8) 재체크 — 갈망 변화 확인\n\n### Screen: “지금은 조금 달라졌나요?”\n\n- 타이틀:\
  \ **“10분 후, 다시 한 번만”**\n\n- 서브: **“내려갔으면 좋고, 안 내려갔어도 괜찮아요.”**\n\n**입력**\n\n- 라벨:\
  \ **“지금 먹고 싶은 마음(갈망)”**\n\n- 스케일 0\\~10\n\n**초간단 보상 선택(선택 1개)**\n\n- 질문: **“지금 가장\
  \ 가까운 느낌은?”**\n\n- 칩: `조금 가벼움` `조금 편안함` `그대로` `잘 참았다` `모르겠음`\n\n**CTA**\n\n- 버튼:\
  \ **“마무리”**\n\n---\n\n# 9) 칭찬 + 다음 액션 추천 (AI처럼 보이게)\n\n### Screen: “오늘의 자동 패턴, 1cm만\
  \ 바꾼 날”\n\n- 타이틀(칭찬): **“좋아요. 자동 스위치를 ‘알아차린’ 게 제일 큰 변화예요.”**\n\n- 본문: **“오늘은 ‘시작\
  \ 버튼’ 전에 한 번 멈췄어요.”**\n\n**다음 액션 1개 카드(딱 1개만)**\n\n- 추천: **“집 도착 전 2분 더 리셋”** 또는\
  \ **“배달앱 열기 전에 물 먼저”**\n\n- 버튼: `이거 해볼래요`\n\n**종료**\n\n- `끝내기`\n\n(대안 칭찬 톤)\n\n\
  - **“완벽이 아니라 반복이 바꿔요.”**\n\n- **“오늘은 ‘자동’ 대신 ‘선택’을 한 번 만들었어요.”**\n\n---\n\n## 핵심\
  \ UX 포인트(딱 3개)\n\n1. “폭식/위기” 대신 **“패턴/자동 스위치/리셋”** 언어로 수치심 제거\n\n2. 트리거 순간 문구는 **“10초만/한\
