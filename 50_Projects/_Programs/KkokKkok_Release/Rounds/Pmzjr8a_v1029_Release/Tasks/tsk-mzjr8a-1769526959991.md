  \ 금지)\n\n- 고급 개인화/모델링\n\n- 회복 카드(다음날) 구현(별도 PRD)\n\n---\n\n## 3. 대상(파일럿 코호트)\n\n\

  \ 1탭 피드백(선택)\n\n---\n\n## 5. 기능 요구사항(FR)\n\n- FR1) reset opt-in on/off\n\n- FR2)\
  \ 알림 슬롯 2개 제한\n\n- FR3) 주 5회 상한 강제\n\n- FR4) 원탭 OFF / 이번주만 끄기 즉시 반영\n\n- FR5) 알림\
