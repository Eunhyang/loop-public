# sosi - 타이머 UI 개선: 물 차오르는 애니메이션 + 마지막 한입 UX

## 설명

### 🎯 아이디어 요약
1. **물 차오르는 애니메이션**: 기존 바깥 초록색 원형 프로그레스 → 내부에서 투명하게 물처럼 차오르는 형태로 변경
2. **마지막 한입 UX**: 물이 다 차오르면 "마지막 한입?" UI 표시하여 자연스러운 식사 종료 유도

### 📂 관련 파일
- `lib/presentation/timer/page/chewing_timer_page.dart` (메인 페이지)
- `lib/presentation/timer/widgets/timer_circle_widget.dart` (원형 타이머 UI)
- `lib/presentation/timer/widgets/timer_painter.dart` (CustomPainter - arc 렌더링)
- `lib/presentation/timer/widgets/timer_animation_manager.dart` (애니메이션 관리)

### 📊 현재 구현
- 외부 원 (200px): 전체 식사 시간 초록색 arc
- 내부 원 (180px): 현재 한입 시간 초록/주황 arc
- CustomPainter로 원형 progress 렌더링

### 💡 제안 구현 방안
1. Wave animation으로 물결 효과 구현
2. 아래에서 위로 차오르는 fill 애니메이션
3. 투명 + 초록색 그라데이션
4. 100% 도달 시 "마지막 한입?" 오버레이 또는 하단 버튼 표시

### 🤔 고려사항
- 한입 시간 vs 전체 식사 시간 중 어떤 것을 물로 표시할지 결정 필요
- Good/Bad 판정 피드백 색상 처리 방안
- 기존 3가지 모드(심플/기본/고급)와의 호환성
