# 폭식 페인킬러 유입은 비타민 유입보다 활성화·리텐션이 높아 DAU를 누적 증가시킨다

> Track: [[Track_1_Product]] | ID: `hyp-1-12` | 상태: planning

## 가설

**Q: 폭식(통제 붕괴)을 겪는 사용자들이 폭식 직전·직후·다음날 상황에서 재발을 막기 어렵다; 이때 폭식 개입(페인킬러) 메시지로 유입된 사용자가 타이머(비타민) 메시지로 유입된 사용자보다 4주 내 활성화와 리텐션이 증가하고 DAU 바닥선이 상승하는가?**

---

## 판정 기준

### 성공 (Success)
4주 관찰(각 코호트 installs>=100, Branch 캠페인 태그 기반 코호트 분리)에서 아래 3개를 모두 충족한다.
- Activation: 페인킬러 코호트 Activation(설치 후 24시간 내 핵심행동 1회 완료) >= 비타민 코호트의 1.5배
- Retention: 페인킬러 코호트 D7 retention >= 비타민 코호트의 1.5배
- DAU 누적: 주간 최소 DAU(관찰 4주 중 최저 주간 평균 DAU) >= 직전 4주 대비 30% 상승

핵심행동(Activation) 정의(운영 표준, 1개 이상 만족):
- 회복 루틴 1회 완료
- 폭식/통제붕괴 에피소드 기록 1회 완료
- 직전 60초 리셋(또는 개입 세션) 1회 완료

### 전환 (Pivot)
6주 관찰(각 코호트 installs>=100)에서 아래 3개를 모두 만족한다.
- Activation: 페인킬러 코호트 Activation <= 비타민 코호트의 1.1배
- Retention: 페인킬러 코호트 D7 retention <= 비타민 코호트의 1.1배
- DAU 누적: 주간 최소 DAU 상승폭 < 직전 4주 대비 10%

## 측정 방법

**데이터 소스**: Branch(캠페인별 유입 태그, installs/opens) + Amplitude 또는 내부 이벤트 로그(Activation, D1/D7) + 앱 DAU 지표(플랫폼 분석)

**측정 지표**:
- Primary:
  - Activation(설치 후 24시간 내 핵심행동 완료율)
  - D7 retention(설치 7일차 재방문/활성 비율)
  - 주간 최소 DAU(주간 평균 DAU 중 최저값)
- Guardrail:
  - 알림 허용률(옵트인 비율) 급락 여부
  - 세션 드롭오프(개입 플로우 첫 30초 이탈) 급증 여부
  - 코어 기능(타이머) 사용률 하락 여부

**코호트 정의(운영 표준)**:
- Painkiller cohort: 폭식 개입 캠페인(예: hyp-3-04 관련 Branch 캠페인)에서 유입된 installs
- Vitamin cohort: 타이머/천천히먹기 캠페인(예: kimmonyang 및 타이머 관련 Branch 캠페인)에서 유입된 installs
- 비교는 동일 기간 동일 플랫폼 기준으로 수행한다.

## 전환 시나리오

### 영향
- 페인킬러 메시지 유입이 클릭/설치만 만들고, 실제 제품 사용(Activation/Retention)으로 이어지지 않을 수 있음
- DAU 누적 성장 엔진으로는 “콘텐츠 웨지” 단독이 부족할 수 있음

### 대응 (피벗/중단)
- 코호트 품질 개선: 페인킬러 유입 온보딩을 “1분 회복/리셋” 중심으로 단순화하고 Activation 정의를 만족시키는 경로를 1스크린으로 축소
- 채널 전환: 콘텐츠 유입 대신 코칭/커뮤니티/검색(의도 높은 유입)에서 동일 코호트 비교를 재수행
- 메시지 재정의: ‘폭식’ 직접 언급이 저항을 만든다면 ‘회복/리셋/자책 차단’ 언어로 재프레이밍 후 재검증
