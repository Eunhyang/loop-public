# LOOP Obsidian Vault

Inner Loop OS (ILOS) 온톨로지 학습 및 스키마 개발을 위한 Obsidian vault입니다.

## 프로젝트 개요

Inner Loop OS는 인간의 정서-섭식-습관-보상-신경계 루프를 하나의 시스템으로 다루는 행동 OS입니다.

## 구조

- `00_Inbox/` - 임시 메모, 아이디어 수집
- `10_Study/` - 온톨로지 학습 노트
- `20_Ontology/` - ILOS 스키마 개발
- `30_Strategy/` - 전략 문서
- `40_LOOP_OS/` - LOOP OS 시스템 정의
- `50_Experiments/` - 실험 및 검증
- `90_Archive/` - 아카이브

## 시작하기

1. Obsidian으로 이 폴더를 vault로 열기
2. `_HOME.md`에서 시작
3. MOC 파일들을 통해 네비게이션

## 핵심 문서

- **스키마**: `20_Ontology/Schema/v0.1/Ontology-lite v0.1 (ILOS) — 규칙(4조건) + 최소 엔티티 스펙.md`
- **시스템 스펙**: `40_LOOP_OS/Inner Loop OS 정의v1.md`
- **개발 가이드**: `CLAUDE.md`
