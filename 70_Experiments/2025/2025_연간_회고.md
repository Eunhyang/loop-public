# 소식연구소 2025년 타임라인별 '해온 일 + 성과 + 회고'

---

# 3월 — 사업 시작 & 초기 문제 정의

### 해온 일

- 개인 경험(15년 요요·폭식·섭식장애) 기반 문제 정의
- 유튜브 "탈다이어트 소식좌" 콘텐츠 시작
- 시청자 피드백 기반 페인포인트 수집
- 초기 페르소나 설정: *"MZ 여성 + 다이어트 앱 경험자"*
- MVP 웹사이트 제작 및 클릭률 테스트(CTR 16%)

### 성과

- 유튜브 초기 구독자 확보
- 콘텐츠를 통한 고객 니즈 확인 (폭식·감정적 섭식 고민 대규모 존재)
- **시장의 존재와 수요 검증 성공**

### 회고

- "식단관리나 칼로리 기록"이 아니라 **'감정–행동–습관 루프'가 핵심 문제**임을 확신
- 콘텐츠 기반 유입이 예상을 훨씬 상회함 → CAC 0원 구조 가능성 발견
- 초기 페르소나는 너무 좁았음

---

# 4월 — MVP 수요검증 & 고객 연구

### 해온 일

- MVP 웹사이트 유입 실험
- 시청자 인터뷰 및 댓글 데이터 기반 니즈 분석
- '왜 실패하는가'에 대한 행동·감정 루프 분석 시작

### 성과

- CTR 16% → 명확한 문제인식 + 솔루션 탐색 의지 높은 사용자층 확인
- "소식좌" 캐릭터 기반 초기 브랜드 정체성 형성

### 회고

- 광고 없이 '문제 → 해결 → 관심'이 이어지는 **유기적 콘텐츠 PMF 초기 신호** 감지
- 데이터 기반 접근 필요성 증가 → 추후 Loop OS 방향성 시드 확보

---

# 5월 — 앱 '꼭꼭' 정식 출시

### 해온 일

- iOS/Android 동시 출시
- 핵심 기능 론칭:
    - 20분 소식 타이머
    - 한입 간격 타이머
    - 감정·포만감 기록
    - 식후 감정일기 루틴
- 유튜브 기반 자연유입 진행

### 성과

- **출시 1개월 만에 누적 가입자 ~2,800명 달성**
- **광고 0원 → 자연유입 100%**
- 앱 DAU 200명대 도달

### 회고

- 기능보다 **브랜드 메시지·콘텐츠·스토리텔링**이 강한 유입 동기가 됨
- 앱 자체의 리텐션은 초기형 기능 대비 매우 양호(초기 Day1 ~50%)

---

# 6월 — 1:1 코칭 '꼭꼭Fit' 론칭

### 해온 일

- 코칭 MVP 텍스트 버전 → 대면 1:1로 피벗
- 1·2기 런칭
- 강의 + 과제 + 피드백 구조 정립
- 웹사이트 개설 & 결제 구조 정비

### 성과

- **2기까지 누적 매출 약 830만 원**
- 코칭 만족도 평균 **4.7/5.0**
- "추천하겠다" 100%
- 페르소나 실제 확인:
    - 30대 전문직
    - 해외 거주 여성
    - 정서 기반 폭식 경험자
- 앱 DAU 300~400으로 상승

### 회고

- 초기 페르소나 가설은 틀렸다 → 실제 고객군은 *"몸보다 감정과 습관 문제를 해결하고 싶은 고가 고객"*
- 고가 상품(10만 원대 이상) 결제 의지 충분
- 코치=현장 라벨러(human-in-the-loop) 역할 가능성 확인
- **코칭 → 앱 → 콘텐츠의 선순환 구조가 형성되기 시작한 시점**

---

# 7~8월 — 중간발표 시점 / 콘텐츠 폭발 성장기

### 해온 일

- 콘텐츠 30건 이상 제작
- 제주 회복 콘텐츠 제작(회복 루틴·요가)
- 상표권 2건 출원
- SNS 마케팅·오픈채팅 커뮤니티 운영

### 성과

- **유튜브 구독자 약 4,000명 / 조회수 총 60만 회↑**
- 앱 가입자 3,200명 돌파
- 코칭 참여자 증가 → 총 매출 약 1,000만 원 돌파
- 콘텐츠 제작 달성률 150%
- 팀 확장(트레이너·편집자·조직 5명 규모)

### 회고

- "헬시플레저"는 이미 오래된 트렌드 → 브랜드 각도 수정
- 제주 로컬 콘텐츠는 감정·회복 메시지 전달에 강력한 임팩트
- 코칭의 니즈가 예상보다 훨씬 강함 → 앱 기능 개선 속도보다 코칭 고도화 우선 필요

---

# 9~10월 — Loop OS 기반 및 고객군 재정의

### 해온 일

- Loop Schema v0.3 → v0.5 확장
- 고객 행동데이터 분석
- 해외 고객군(싱가폴·미국 등) 유입 증가
- 정신과 전문의·AI 시니어 자문단 구두 합류
- 구독 모델 실험(AI 리포트·와치 연동)

### 성과

- 앱 가입자 **5,900명 → 6,000명 이상**
- 유입의 95% 이상 콘텐츠 기반(광고 비용 제로)
- 30~40대 전문직 비중 지속 증가 → 고가 전환율 높아짐
- 글로벌 확장 가능성 확인(해외 거주자 결제 의지 강함)

### 회고

- '다이어트 앱' 시장보다 → **정서·폭식·행동 루프 OS 시장**이 훨씬 큰 기회
- 2026년 유료화 전략과 GLP-1 Behavioral 시장 공략 필요
- 콘텐츠 → 앱 → 코칭 퍼널이 구조적으로 작동하기 시작

---

# 11월 — 생태계 완성 단계 진입

### 해온 일

- 구독 서비스 출시
- 와디즈 준비
- 앱 리텐션 및 데이터 정제
- 코칭 4기까지 운영(누적 매출 약 1,850만 원)
- 팀 5명 운영 안정화
- 고밀도 사용자층(Tier1·2) 약 200~300명 확보

### 조직 고도화(중요 업데이트)

- **데이터 리드(Data Lead) 합류 → Loop Schema/행동 패턴 분석 체계 확립**
- **정신과 전문의 Senior Advisor 영입 → 정서·섭식·폭식 루프 임상 검증 라인 구축**

### 성과

- **앱 DAU 300~400 유지**
- 앱 리텐션 day1 50% day30 19% 달성
- **콘텐츠 누적 조회수 80만~84만**
- **코칭 총 50명 이상 참여 / 매출 1,850만 원**
- **정서 기반 루프 패턴 데이터 축적**
- 유튜브 기반 유입 구조 안정화
- **와디즈 선런칭(11월)** → 4일 만에 **알림신청 320명 돌파**

### 회고

- 광고를 거의 쓰지 않았음에도 유입이 꾸준 → CAC 0원 기반의 매우 강한 성장 잠재력
- 코칭의 전환율이 앱 대비 매우 높음 → 코칭을 프리미엄 수익모델로 확정
- 핵심 고객층이 분명함 → 2026년에는 '해외 전문직 여성·폭식 회복군' 중심 퍼포먼스 마케팅 필요
- Loop OS v1을 위한 데이터 구조 정비 필요

---

# 2025년 종합 회고 (핵심 6줄)

1. **문제가 아니라 고객을 발견한 해였다.**
    (폭식·정서 기반 식사 문제군이 명확히 드러남)

2. **앱·콘텐츠·코칭이 하나의 '회복 루프'로 연결되는 생태계가 형성되기 시작했다.**

3. **초기 페르소나는 틀렸지만 실제 결제 고객군을 명확하게 찾아냈다.**

4. **광고 없이도 성장하는 구조(CAC Zero)를 증명했다.**

5. **코칭의 고가 전략이 가능하다는 것을 시장이 먼저 보여줬다.**

6. **Loop OS(감정·습관·섭식 루프)라는 장기 전략의 실현 가능성이 생겼다.**

---

# 2026년을 위한 전략적 결론 (What's Next)

- **앱 완전 유료화 + GLP-1 Behavioral 콘텐츠 선점**
- **고밀도 사용자 2,000명 확보**
- **프리미엄 코칭(35만 원) 고도화 및 코치팀 확장**
- **콘텐츠 기반 퍼포먼스 마케팅 시작**
- **해외 고객군 타기팅(한인·전문직 시장)**
- **Loop OS v1 구축 위한 데이터 라벨링 강화**
