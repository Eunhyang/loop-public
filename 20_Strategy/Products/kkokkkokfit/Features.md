# 꼭꼭FIT - 현재 기능

> 기준: 4기 기준

---

## 웹 플랫폼 기능

### 1. 음식 편향 테스트 (Food Bias Test)
- 개인 식습관 패턴 진단
- 편향 유형 분석 및 결과 제공
- 코칭 프로그램 진입점

### 2. 온라인 강의 플랫폼
- 12주차별 영상 강의
- 섭식 심리 이론 + 행동 변화 기법
- 주차별 과제 페이지

### 3. 결제 시스템
- Toss Payments 연동
- 와디즈 펀딩 연계

### 4. CoachOS (AI 코칭 시스템)
- GPT-4.1 기반 AI 피드백
- 식사 일기 자동 분석
- 코치용 대시보드
- 고객별 장기 요약 관리

### 5. 인증 시스템
- Google / Kakao 소셜 로그인
- 코치 전용 로그인 (email/password)

---

## 프로그램 구성

### 1. 1:1 밀착 코칭
- 전담 코치 배정
- 매일 데이터 기반 피드백
- 주 1회 화상 세션

### 2. 꼭꼭앱 연동
- Pro 기능 전체 제공
- 식사 일기 + 감정 기록 추적
- 코치가 실시간 데이터 확인

### 3. 섭식 행동 분석
- "언제, 왜 먹는지" 패턴 파악
- 감정-폭식 트리거 식별
- 주차별 분석 리포트

### 4. 행동 교정 과제
- 주차별 맞춤형 미션
- 구체적 행동 단위 실천
- 진행 상황 체크

---

## 12주 커리큘럼 개요

| 구간 | 주차 | 핵심 내용 |
|------|------|----------|
| 진단 | 1-2주 | 현재 식습관 패턴 분석 |
| 인식 | 3-5주 | 감정-식사 연결고리 인지 |
| 변화 | 6-9주 | 대안 행동 학습 및 실천 |
| 정착 | 10-12주 | 새 습관 안착 및 유지 전략 |

| 항목 | 내용 |
