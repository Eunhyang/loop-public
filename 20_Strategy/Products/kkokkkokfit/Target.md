# 꼭꼭FIT - 타겟 고객

---

## 핵심 타겟

**"다이어트 여러 번 해봤지만 폭식과 요요로 매번 실패, 이번엔 근본 원인을 해결하고 싶은 사람"**

---

## 페르소나

### Primary: 반복 실패자
| 항목 | 내용 |
|------|------|
| 연령 | 25-45세 여성 |
| 특징 | 다이어트 3회 이상 시도, 모두 요요 |
| Pain Point | "왜 나만 계속 실패할까?" |
| 니즈 | 근본 원인 규명, 전문가 1:1 케어 |
| 지불 의향 | 높음 (문제 해결 절박) |

### Secondary: 감정적 폭식자
| 항목 | 내용 |
|------|------|
| 연령 | 20-35세 |
| 특징 | 스트레스/감정 변화 시 폭식 반복 |
| Pain Point | "먹고 나서 후회하는 사이클" |
| 니즈 | 폭식 트리거 인지, 대안 행동 학습 |
| 지불 의향 | 중-높음 |

### Tertiary: 프리미엄 케어 추구자
| 항목 | 내용 |
|------|------|
| 연령 | 30-50대 전문직 |
| 특징 | 시간 없음, 혼자 관리 어려움 |
| Pain Point | "비용보다 결과가 중요" |
| 니즈 | 밀착 관리, 데이터 기반 코칭 |
| 지불 의향 | 매우 높음 |

---

## 타겟 vs 앱 고객 비교

| 구분 | 꼭꼭앱 (Free/Pro) | 꼭꼭FIT |
|------|-------------------|---------|
| 셀프 관리 | 가능 | 어려움 |
| 문제 심각도 | 경-중 | 중-심 |
| 지불 의향 | 낮음-중간 | 높음 |
| 접근 방식 | 앱 혼자 사용 | 1:1 밀착 코칭 |
