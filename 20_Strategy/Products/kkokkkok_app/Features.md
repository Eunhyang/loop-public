# 꼭꼭앱 - 현재 기능

> 기준 버전: v1.0.28

---

## 핵심 기능

### 1. 20분 소식 타이머
- 식사 시작 시 타이머 시작
- 천천히 씹기 유도
- 완료 시 식사 기록으로 연결

### 2. 식사 일기 (Meal Diary)
- **식전**: 배고픔 정도 (1-10)
- **식중**: 타이머 + 천천히 먹기
- **식후**: 포만감 정도 (1-10), 감정 기록

### 3. 감정 기록
- 식사 전후 감정 상태 선택
- 감정-식습관 패턴 분석 기반 데이터

### 4. 루틴 관리
- 매일 체크할 습관 설정
- 완료 체크 및 연속 달성 추적

### 5. AI 코칭 (Pro)
- Gemini 기반 대화형 코칭
- 식습관 패턴 분석 피드백
- 맞춤형 조언 제공

### 6. 인사이트 (Pro)
- 주간/월간 식습관 데이터 분석
- 감정-식사 패턴 시각화
- 개선 포인트 제안

---

## 기술 특징

| 항목 | 내용 |
|------|------|
| 아키텍처 | Local-First + Outbox Pattern |
| 오프라인 | 완전 지원 (Drift DB) |
| 동기화 | Firebase 연동 (온라인 시) |
| 결제 | RevenueCat |
| 분석 | Amplitude |
