# 꼭꼭앱 - 타겟 고객

---

## 핵심 타겟

**"다이어트를 반복해도 폭식과 요요를 멈추지 못하는 사람"**

---

## 페르소나

### Primary: 요요 반복자
| 항목 | 내용 |
|------|------|
| 연령 | 20-40대 여성 |
| 특징 | 다이어트 성공 → 폭식 → 요요 사이클 반복 |
| Pain Point | "왜 나는 의지가 약할까?" |
| 니즈 | 근본 원인 파악, 지속 가능한 해결책 |

### Secondary: 감정적 섭식자
| 항목 | 내용 |
|------|------|
| 연령 | 20-30대 |
| 특징 | 스트레스/감정 변화 시 폭식 |
| Pain Point | "기분이 안 좋으면 먹게 된다" |
| 니즈 | 감정-식습관 연결고리 인지, 대안 행동 |

### Tertiary: GLP-1 복용자
| 항목 | 내용 |
|------|------|
| 연령 | 30-50대 |
| 특징 | 위고비 등 GLP-1 약물 복용 중/중단 |
| Pain Point | "약 끊으면 다시 폭식이 터진다" |
| 니즈 | 약물 중단 후 습관 유지, 정서적 케어 |

## 시장 세그먼트

| 세그먼트 | 규모 | 접근 전략 |
|----------|------|-----------|
| 국내 다이어트 실패 경험자 | - | 콘텐츠 마케팅 + 인플루언서 |
